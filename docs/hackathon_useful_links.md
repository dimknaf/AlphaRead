## CityFalcon (data trigger — our news source)

- API base: `https://api.cityfalcon.com/v0.2`
- Stories endpoint: `https://api.cityfalcon.com/v0.2/stories?identifier_type=assets&identifiers=<TICKER>&access_token=<KEY>`
- Similar stories: `https://api.cityfalcon.com/v0.2/stories/{uuid}/similar_stories`
- Entity sentiment: `https://api.cityfalcon.com/v0.2/services/sentiment`
- DCSC base: `https://api.cityfalcon.com/dcsc/v0.1`
- DCSC sector classification (ticker → sectors): `https://api.cityfalcon.com/dcsc/v0.1/portfolio_classification`
- DCSC adjacent companies (sectors → companies): `https://api.cityfalcon.com/dcsc/v0.1/smart_portfolio`
- Public product page: [cityfalcon.ai/products/api/financial-news](https://www.cityfalcon.ai/products/api/financial-news)
- Pricing: [cityfalcon.ai/products/api/financial-news/pricing-api/commercial](https://www.cityfalcon.ai/products/api/financial-news/pricing-api/commercial)
- Gated full docs: [dev.cityfalcon.com/doc/api/v0.2](https://dev.cityfalcon.com/doc/api/v0.2)
- Help centre: [help.cityfalcon.com](http://help.cityfalcon.com/)
